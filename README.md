# fileReader

A media file browser application built with Nextron (Next.js + Electron).

<p align="center"><img src="https://i.imgur.com/a9QWW0v.png"></p>

## Development

### Install Dependencies

```bash
# using yarn
$ yarn

# or using npm
$ npm install
```

### Development Mode

```bash
# using yarn
$ yarn dev

# or using npm
$ npm run dev
```

### Production Build

```bash
# using yarn
$ yarn build

# or using npm
$ npm run build
```
